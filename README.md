# 职工管理系统 java923

## 项目概述

职工管理系统，又称员工管理系统或人事管理系统，是一款基于前后端分离模式开发的应用。它主要用于协助企业进行人事管理，提高工作效率。

## 技术栈

- 开发语言：Java
- 后端框架：SpringBoot
- 数据持久层：MyBatis-Plus
- 前端框架：Vue
- 开发工具与依赖：
  - IDEA
  - JDK1.8
  - Maven
  - Node.js

## 数据库

- MySQL

## 功能模块

### 用户模块
- 用户登录

### 部门管理
- 添加部门
- 部门列表

### 员工管理
- 添加员工
- 员工列表

### 考勤管理
- 添加考勤
- 考勤审核列表

### 奖惩管理
- 添加奖惩
- 奖惩列表

### 合同管理
- 添加合同
- 合同列表

## 运行环境

- 操作系统：支持Windows、Linux等
- 数据库：MySQL（版本要求5.7及以上）
